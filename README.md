# Node.js Projects Collection

Collection of Node.js projects, APIs and studies developed while learning backend development concepts and best practices.

## Projects

* **Bootcamp-Helpper-Origins** - Bootcamp exercises and helper utilities
* **Clean-node-api** - API following clean architecture principles
* **Node-api-mongodb-crud** - CRUD API using MongoDB and TypeORM
* **Node-api-mysql-crud** - CRUD API using MySQL and TypeORM
* **Node-full-api-riott** - Complete REST API project
* **Node-html-pdf** - HTML to PDF generation with Node.js
* **Project-ejs-node-old** - Server-side rendering with EJS

## Technologies

* Node.js
* TypeScript
* Express
* MongoDB
* MySQL
* TypeORM
* Docker
* Jest
* Swagger
* EJS

## Purpose

This repository serves as a collection of backend projects, experiments and studies covering API development, databases, testing, architecture patterns and server-side applications.
